# App Icon Design

## Summary

Add a programmatic app icon for MarkdownQuickLook — a 1024x1024 PNG generated by a Swift script using CoreGraphics.

## Requirements

- 1024x1024 full-bleed PNG (macOS applies squircle mask)
- Background: linear gradient top-to-bottom, dark lavender (#3D2A5C) to light lavender/near-white (#E8E0F0)
- Line 1: ".md" in SF Mono Bold, ~340pt, white, centered horizontally, upper-center
- Line 2: "QuickLook" in SF Mono Regular, ~120pt, white, centered horizontally, below line 1
- Generated by a one-shot Swift script, not a build phase
- Output wired into asset catalog so Xcode uses it as the app icon

## Implementation

**Generator script:** `Scripts/generate-icon.swift` — standalone Swift script using CoreGraphics and CoreText to render the icon PNG. Invoked via `swift Scripts/generate-icon.swift`.

**Output path:** `MarkdownQuickLookApp/Assets.xcassets/AppIcon.appiconset/icon_1024x1024.png`

**Asset catalog:** Create `MarkdownQuickLookApp/Assets.xcassets/AppIcon.appiconset/Contents.json` pointing to the single 1024x1024 image. macOS automatically derives all needed sizes.

**project.yml:** Add `MarkdownQuickLookApp/Assets.xcassets` to the app target's sources and set `ASSETCATALOG_COMPILER_APPICON_NAME: AppIcon`.

## Files Changed

| File | Change |
|------|--------|
| `Scripts/generate-icon.swift` | New: Swift script to generate the icon PNG |
| `MarkdownQuickLookApp/Assets.xcassets/AppIcon.appiconset/Contents.json` | New: asset catalog metadata |
| `MarkdownQuickLookApp/Assets.xcassets/AppIcon.appiconset/icon_1024x1024.png` | New: generated icon image |
| `project.yml` | Add asset catalog source path and app icon build setting |

## Out of Scope

- Multiple icon sizes (macOS derives them from 1024x1024)
- Build-phase icon generation (one-shot script only)
- Extension icon (extension inherits from host app)
